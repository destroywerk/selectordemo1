## Getting Started

1. Clone the repository:
   ```bash
   git clone [your-repo-url]
   cd people-selector
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
